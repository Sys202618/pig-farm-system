# 猪场生产+财务管理系统

简洁实用的猪场一体化管理系统，支持生产数据管理和简易财务成本核算。

## 系统特点

- **简单实用**：界面简洁，操作直观，无需专业培训
- **一体化管理**：生产数据和财务数据在同一系统
- **权限分明**：管理员/生产员/财务 三种角色
- **轻量部署**：Python + SQLite，无需额外数据库

## 功能模块

### 生产管理
| 模块 | 功能 |
|------|------|
| 猪场信息 | 猪场增删改查 |
| 猪舍管理 | 猪舍类型、容量管理 |
| 员工管理 | 员工信息、职位管理 |
| 猪群管理 | 批次、猪只类型、数量 |
| 配种记录 | 配种日期、母猪号、方式、预产期 |
| 产仔记录 | 产仔日期、总仔/活仔/死胎数 |
| 断奶记录 | 断奶日期、断奶头数、均重 |
| 转舍记录 | 猪只转移记录 |
| 死亡记录 | 死亡/淘汰记录 |
| 销售记录 | 销售数量、重量、金额 |

### 财务报表
| 模块 | 功能 |
|------|------|
| 存栏统计 | 各类型猪只存栏数量 |
| 产仔统计 | 月度产仔窝数、活仔数 |
| 断奶统计 | 月度断奶情况 |
| 销售统计 | 月度销售收入 |
| 死亡统计 | 月度死亡淘汰统计 |
| 月度收支 | 收入-成本=盈亏 |
| 成本汇总 | 8大类成本明细 |
| 头均成本 | 总成本/头数 |

## 成本分类（8大类）
1. 饲料成本
2. 兽药疫苗成本
3. 猪精成本
4. 人工成本
5. 水电费用
6. 折旧费用
7. 维修费用
8. 其他费用

## 成本计算公式
```
总成本 = 所有费用相加
头均成本 = 总成本 ÷ 销售/存栏头数
```

## 快速启动

### 方式一：双击启动
双击 `启动系统.bat`

### 方式二：命令行
```bash
cd pig_farm_system
pip install flask flask-cors openpyxl
python backend/app.py
```

### 方式三：Docker（可选）
```bash
docker build -t pig-farm .
docker run -p 5000:5000 pig-farm
```

## 访问系统

- 地址：http://127.0.0.1:5000
- 用户名：admin
- 密码：admin123

## 角色权限

| 角色 | 生产管理 | 财务查看 | 系统设置 |
|------|---------|---------|---------|
| 管理员 | 全部 | 全部 | 全部 |
| 生产员 | 增删改查 | 不可见 | 不可见 |
| 财务 | 只读 | 全部 | 不可见 |

## 数据库

- 使用 SQLite，无需安装
- 数据库文件：`data/pig_farm.db`
- 首次启动自动创建

## 注意事项

1. 首次使用请先在"猪场信息"中添加猪场
2. 财务数据需要按月录入
3. 报表查询需要选择猪场和月份
4. 导出Excel功能支持多种报表
