# 猪场数据管理与成本核算一体化系统 - 完整规格说明

## 一、系统概述

- **系统名称**：猪场数据管理与成本核算一体化系统
- **系统定位**：B/S架构，浏览器访问，支持生产管理与成本核算两大核心模块
- **适用对象**：集团管理层、财务部门、生产部门、场级管理人员
- **数据层级**：集团 → 区域 → 子公司 → 场线 → 场次 → 部门 → 工段

## 二、工段定义（固定）

- 仔猪（待产）
- 保育
- 育肥
- 后备

## 三、权限角色

| 角色 | 说明 |
|------|------|
| super_admin | 超级管理员，可管理所有数据 |
| finance | 财务，可查看/录入成本数据 |
| production | 生产，可录入/查看生产数据 |
| dept_view | 部门查看，仅查看本部门数据 |

## 五、核心计算公式

### 5.1 销售成本

```
均重 = 销售总重量 ÷ 销售头数
料价 =（前阶段饲料成本 + 本阶段饲料成本）÷（前阶段料重 + 本阶段料重）
料比 =（前阶段料重 + 本阶段料重）÷ 销售总重量
公斤成本 = 对应科目成本 ÷ 销售重量
头均成本 = 对应科目成本 ÷ 销售头数
```

### 5.2 仔猪出生成本

```
配准率 =（配种头数 − 返情 − 检空）÷ 配种头数
分娩率 =（分娩母猪 + 怀孕销售）÷ 配种头数
窝均健仔 = 健仔数 ÷ 产仔母猪数
仔猪头均出生成本 = 待产转出成本 ÷ 活仔数
```

### 5.3 断奶成本（校正体重）

```
断奶成本 = 实际头均成本 +（校正体重 − 实际均重）× 哺乳段单位增重成本
校正体重：6kg、7kg
统计范围：转出 + 内销 + 外销，剔除同段转群
```

### 5.4 保育转出成本（校正体重）

```
保育成本 = 实际头均成本 +（校正体重 − 转出均重）× 保育单位增重成本
校正体重：28kg、29kg、30kg
统计范围：转出 + 内销，剔除同段转群
```

### 5.5 增重成本

```
料肉比 = 料重 ÷ 增重
含死亡增重成本 = 生产成本 ÷ 总增重
不含死亡增重成本 = 生产成本 ÷（总增重 − 死亡增重）
```

### 5.6 精液成本

```
公猪平均存栏 = 饲养头日数 ÷ 天数
精液份数 = 场内接收 + 外销
头均精液份数 = 总份数 ÷ 平均存栏
```

### 5.7 种猪日成本

```
日成本 = 当期成本 ÷ 饲养头日数
```

## 六、功能模块

### 模块A：生产数据管理

- 基础信息管理（猪舍、猪群、批次、耳号、部门、人员、物料）
- 生产数据录入（配种、分娩、断奶、转群、销售、死亡、淘汰、库存）
- 数据自动汇总（头数、重量、日龄、均重、料重、采食量）
- 生产报表（配种报表、分娩报表、断奶报表、转群报表、销售报表）
- 数据导入导出（Excel批量导入、自动校验）

### 模块B：成本核算管理

- 销售成本管理
- 仔猪出生成本管理
- 断奶成本管理
- 保育转出成本管理
- 增重成本管理
- 精液成本管理
- 种猪日成本管理
- 商品肉猪成本分析
- 头均转出成本排名

## 七、筛选功能

- 工段筛选：仔猪、保育、育肥、后备
- 月份筛选：单月 / 多月平均
- 部门筛选：按部门、场线、子公司、区域
- 销售类型：外销、内销肉食、全部
- 单位切换：公斤成本 / 头均成本 / 日成本

## 八、预警规则

- 料比超标（>3.5）：标红预警
- 公斤成本超标：标红预警
- 日龄超标：标红预警
- 异常数据自动标记
